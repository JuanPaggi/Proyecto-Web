# Proyecto-Web

Proyecto Web con Backend y Frontend

## Frontend:

### Intalar angular:

- npm install -g @angular/cli

### Actualizamos las dependencias:
- npm install

#### Crear Proyecto
- ng new nombre

#### Crear componente
- ng generate component nombre

#### Crear servicio
- ng generate service nombre

#### Levantar proyecto
- ng serve -o

#### Compilar Proyecto
- ng build
